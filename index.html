Interactive CLDN1 Correlation Plots

Interactive Spearman R correlation plots in:

  • AT1_CLDN1_positive
  • AT2_CLDN1_positive
  • Aberrant Basaloid_CLDN1_positive

Hover Feature

Each dot respresent a different gene. Hovering over each dot on the plot will reveal; R score, p value, and N cell counts.

Filtering

Filter by N values (min number of cell), p value less than 0.05 or FDR adjusted significance levels.

Example:
An N > 200 and p-value < 0.05 were used to define statistically meaningful R scores. For each cell type, R scores were computed only when at least 200 cells contributed to the calculation.
